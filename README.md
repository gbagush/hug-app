## üèÉ How To Run

- clone repository

  ```bash
  git clone https://github.com/gbagush/hug-app
  ```

- go to directory

  ```bash
  cd hug-app
  ```

- install dependency

  ```bash
  npm i
  ```

- run (on development mode)

  ```bash
  npm run dev
  ```
